## License Summary

## Usage

Each theme file contains metadata in its header:

```vim
## name:     Theme Name
## author:   Author Name
## license:  License Type
## upstream: Original Source URL
## blurb:    Short Description
```

For more information about individual themes, see their upstream repositories linked above.
